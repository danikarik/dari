# DARI

[Registry](http://www.ndda.kz/) parser

## Build

```sh
make
```

or

```sh
make build
```

## Build for Windows

```sh
make build-win
```

## Run in development mode

```sh
make dev
```

> [!NOTE]
> Runs program with limit of 1 page or 300 records.

## Run in production mode

```sh
make prod
```
